## Tag

- **Name:** `v0.3.0-read-model`
- **Type:** annotated (`git tag -a`), message = "P11-A Structured World Read-Model".
- **Target commit:** the **`main` merge commit** of the P11-A PR
  (`feature/p11-structured-world-model` ‚Üí `main`, **PR #6**) ‚Äî *not* a branch tip.

## Scope being tagged

**P11-A Structured World Read-Model only.** Read-model layer; no engine change.

- New: `src/chronicle_forge/reporting/world_model.py`
  (`world_model(world) -> WorldView`, `world_model_json(world) -> str`).
- `WorldView` ‚Äî immutable (`frozen=True, extra="forbid"`), id-free aggregate:
  `schema_version, overview, theme, lives, heritage, bonds, places, factions`.
- `places` introduced structured-first (the 3D map anchor).
- `SCHEMA_VERSION = "1"`, deliberately decoupled from `ENGINE_VERSION`.

**Explicitly NOT in this tag:** any engine change, Recipe change, `ENGINE_VERSION`
bump, P11-B (Social Memory L2), and the `.codegraph/` ignore chore (carried
separately on `feature/chore-ignore-codegraph`, not part of this release).
